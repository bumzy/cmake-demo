# cmake-demo
C++ project demo build with CMake using third party gRPC, gtest and JSON

## Environment
### Ubuntu
 * `sudo apt install cmake build-essential`

## Build
 * cmake version >= 3.5
 * `git clone --recursive ...`
 or `git pull && git submodule update --recursive --init`
 * `mkdir build && cd build`
 * `cmake ..`
 * `make`
